# dragonSlaying
> 屠龙术

技术层次的精进无非有两个方向：

+ 面向纯技术的硬核能力提升 —— 结构、算法、底层、源码……
+ 面向业务的技术解决方案 —— 监控、安全、性能、体验、质保……

而屠龙术就是后者的精华整合，屠龙术中的每一种技术体系都是真真正正能针对业务的某方面做出提升的完整解决方案。
